MFC ĐỘC – Static HTML Website

Cách mở:
1. Giải nén file ZIP.
2. Mở index.html bằng trình duyệt.
3. Các trang chính gồm:
   - index.html: Trang chủ
   - about.html: MFC là gì?
   - journey.html: Hành trình hình thành MFC
   - fashion-show.html: Fashion show là gì?
   - tickets.html: Mua vé
   - seats.html: Sơ đồ ghế
   - checkout.html: Thanh toán
   - recruit.html: Tuyển cộng tác viên

Thư mục assets chứa ảnh logo và key visual được nhúng trực tiếp trong giao diện.
Thư mục mockups chứa các ảnh mockup đã gen trước đó để tham khảo khi dựng Figma hoặc code tiếp.

Lưu ý: Đây là HTML/CSS/JS tĩnh, chưa có backend xử lý thanh toán, lưu form hoặc gửi email.
